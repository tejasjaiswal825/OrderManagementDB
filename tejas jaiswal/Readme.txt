OrderManagementDB Project

This project is divided into three parts:
1. Setting up and populating a MySQL database using XAMPP.
2. Performing data processing and analysis using Python in a Jupyter Notebook.
3. Visualizing the data using Power BI.

Part 1: Setting Up the Database

Requirements
- [XAMPP](https://www.apachefriends.org/index.html)
- MySQL Workbench (optional, for easier database management)

Steps
1. Install XAMPP: Download and install XAMPP from the official website.
2. Start MySQL: Open the XAMPP Control Panel and start the MySQL module.
3. Create the Database:
   - Open phpMyAdmin from the XAMPP Control Panel.
   - Create a new database named `CustomerOrderDB`.
   - Use the following SQL script to create the necessary tables and populate them with sample data.


Part 2: Data Processing and Analysis

Requirements
- Python 
- Jupyter Notebook
- Required Python Libraries: pandas, sqlalchemy, pymysql, numpy, sklearn, openpyxl

Steps
1. Set Up the Environment:
	- Install Code Editer from the official website.
	- Open Anaconda Navigator and launch Jupyter Notebook.
2. Install Required Libraries:
	- Open a new notebook and run the following commands to install the required libraries:
	  pip install pandas sqlalchemy pymysql numpy sklearn openpyxl
